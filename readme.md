# CRM DEPLOY

## Project setup
```
make install
```

### Run migrations
```
make migrate
```

### Make migrations
```
make makemigrations
```

### Start crm
```
make start
```

### Stop crm
```
make stop
```

### Go to bash
```
make shell
```

### Remove containers
```
make destroy
```

### Collect static
```
make collectstatic
```

### Start celery
```
make celery
```
